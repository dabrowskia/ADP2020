2. Ktry kraj posiada najmniejsz liczb ludnoci i ile wynosi?

Najmniejsz liczb ludnoci ma Grenlandia.

world %>%
  arrange(pop) %>%
  select(name_long,pop) %>%
  top_n(1, wt=-pop) %>% 
  st_set_geometry(NULL) %>% 
  kable()
name_long pop

3. Ile krajw znajduje si w Azji?

W Azji znajduje si 47 krajw.

world %>%
  group_by(continent) %>% 
  summarise(count=n()) %>% 
  filter(continent == "Asia") %>% 
  st_set_geometry(NULL) %>% 
  kable()
continent count
Asia 47

4. Wywietlcie histogram powierzchni wszystkich niezalenych krajw (Sovereign Country).

world %>%
  filter(type == "Sovereign country") %>%
  ggplot(aes(area_km2)) +
    geom_histogram(bins = 20) +
    scale_x_log10() +
    labs(title='Powierzchnia krajw niezaleznych',
         x='Powierzchnia w km2',
         y='Liczba krajw') +
    theme_classic()
